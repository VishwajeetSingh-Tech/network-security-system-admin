# 📋 Lab Summary — Network Security & Large Data Solutions

## Lab 1 — Windows Firewall Access Policy

### Objective
Configure firewall rules to restrict unauthorized network access in a Windows environment.

### What I Did
- Set up a Windows 10 virtual machine using VMware
- Created inbound and outbound firewall rules to block ports 80 and 443
- Configured application-specific rules to restrict browser internet access
- Tested and validated all firewall policies to confirm effectiveness

### What I Learned
- How firewall port blocking prevents unauthorized web traffic
- How application-level rules add an extra layer of security
- How to document and verify security policy configurations
- Real-world relevance: hospitals use similar firewall rules to protect patient portals

## Lab 3 — Cloudera Hadoop Big Data Server

### Objective
Install and configure a Big Data infrastructure using Cloudera Hadoop in a virtualized environment.

### What I Did
- Imported and configured Cloudera Hadoop virtual machine
- Set up a single-node Hadoop cluster with Cloudera Manager
- Examined HDFS, MapReduce, and Big Data architecture components
- Installed and secured a traditional database server alongside the Hadoop cluster

### What I Learned
- How Hadoop handles large-scale data storage and processing
- How to configure and manage a Big Data cluster environment
- How traditional databases and Big Data systems can work together
- Real-world relevance: healthcare organizations use Hadoop for processing large patient datasets and clinical analytics
