# 🔐 Network Security & Large Data Solutions

## 📌 Overview
Hands-on lab work covering enterprise network security, cybersecurity 
threat detection, and big data infrastructure setup in virtualized environments.
All labs were completed in simulated enterprise environments using industry-standard tools.

## 🏥 Healthcare IT Relevance
- Firewall policies protect Electronic Health Records (EHR) from unauthorized access
- Phishing detection is the #1 cybersecurity threat in hospital environments
- Hadoop/Big Data skills apply directly to healthcare analytics and patient data processing
- VM administration mirrors real hospital IT infrastructure management

## 🛠️ Labs Completed

### Lab 1 — Windows Firewall Access Policy
- Configured Windows Firewall rules to block ports 80 and 443
- Implemented application-level restrictions to control browser access
- Set up and managed security policies in a virtualized Windows environment
- Validated firewall effectiveness through controlled access testing

### Lab 2 — Phishing Site Detection & Cybersecurity Awareness
- Identified and analyzed real-world phishing websites
- Applied cybersecurity best practices for threat recognition
- Documented security findings and risk assessment recommendations
- Demonstrated awareness of social engineering attack vectors

### Lab 3 — Cloudera Hadoop Big Data Server
- Installed and configured Cloudera Hadoop in a virtualized environment
- Set up a single-node cluster with Cloudera Manager
- Explored Big Data architecture components (HDFS, MapReduce)
- Installed and secured both traditional and Big Data database servers
